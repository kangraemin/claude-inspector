# Phase 6: UI 컴포넌트

## 목표
React UI 컴포넌트를 구현하여 기존 index.html과 동일한 기능을 제공한다.

## 범위
- i18n 유틸리티 (ko/en)
- shared: Spinner, ElapsedTimer, JsonTree
- Header 컴포넌트
- ProxyPanel: ProxyControl + ProxyList
- ProxyDetail: Request/Response/Analysis/AiFlow 탭
- AiFlow: AiFlowPanel + AiFlowResult + Optimization + Chat
- CSS styles (기존 index.html CSS 이관)
- App.tsx에 전체 레이아웃 통합

## Steps
1. ProxyPanel + ProxyList (i18n, shared 컴포넌트 포함)
2. ProxyDetail 탭 (Request/Response/Analysis/AiFlow within detail)
3. AiFlow + Optimization + Chat
4. shared 컴포넌트 + CSS + App 통합
